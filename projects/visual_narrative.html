SUMMARY

This project page demonstrates works under two categories:
 • Calliope: an intelligent system for automatically generating, editing, and sharing visual data stories.
 • Researches that are broadly related to narrative visualization, including infographics, factsheet, animation, data video, annotated chart, and so on.
Keywords: Data-driven Storytelling, Automatic Visual Design, Information Visualization
